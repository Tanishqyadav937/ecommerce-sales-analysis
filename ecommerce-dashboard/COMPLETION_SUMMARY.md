## 🚀 Quick Start (3 Steps)

### Step 1: Install
```bash
# macOS/Linux
bash install_requirements.sh

# Windows
install_requirements.bat
```

### Step 2: Activate
```bash
# macOS/Linux
source venv/bin/activate

# Windows
venv\Scripts\activate.bat
```

### Step 3: Run
```bash
streamlit run app.py
```

**Dashboard opens at**: http://localhost:8501
